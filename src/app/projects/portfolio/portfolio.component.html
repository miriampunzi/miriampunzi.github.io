The very first version of this portfolio was built using Google Sites. I needed a quick and easy solution to start showcasing my projects.

For the second version, I wanted to showcase my programming skills by building a custom website. Creating my own portfolio allowed me to overcome the limitations of Google Sites and have full control over design and functionality.

For the third version, I wanted a more personalized design. After having a color analysis session with an image consultant, I changed the design using colors of the palette that complements my natural tones.
